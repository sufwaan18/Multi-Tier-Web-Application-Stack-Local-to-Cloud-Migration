# Multi-Tier Web Application Stack: Local to Cloud Migration

## Overview

This project demonstrates migration of a production-grade multi-tier Java web application from a local environment to AWS cloud infrastructure.

## Architecture

Internet
|
Nginx Load Balancer
|
Apache Tomcat Application Tier
|
RabbitMQ
|
Memcached
|
MySQL / Amazon RDS

## AWS Services

- Amazon EC2
- Amazon RDS
- Amazon ElastiCache
- AWS EKS
- AWS ECR
- CloudWatch
- IAM
- VPC

## DevOps Tools

- Docker
- Kubernetes
- Terraform
- GitHub Actions

## Features

- Cloud migration strategy
- Infrastructure as Code
- Containerized workloads
- CI/CD automation
- Monitoring and observability
- Disaster recovery and backups

## Skills Demonstrated

AWS, EC2, RDS, ElastiCache, Docker, Kubernetes, Terraform, GitHub Actions, Cloud Migration, DevOps
